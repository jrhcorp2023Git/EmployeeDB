# EmployeeDB

A beginner-friendly SQL project for managing and exploring employee data.

## Files Included
- `setup.sql`: Creates the database schema
- `Add Employee Data.sql`: Inserts sample employee records
- `Explore Employee Data.sql`: Runs queries to analyze data
- `company.db`: SQLite database file

## How to Use
1. Open `company.db` in SQLite.
2. Run `setup.sql` to create tables.
3. Run `Add Employee Data.sql` to insert records.
4. Use `Explore Employee Data.sql` to explore the data.
